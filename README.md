# Patrick's Projects Repository

This repository contains multiple projects:

## Projects

### 1. L2P (Learn to Play)
A comprehensive learning platform with authentication, question sets, and interactive features.

**Tech Stack:**
- Frontend: React, TypeScript
- Backend: Node.js, Express
- Database: PostgreSQL
- Deployment: Docker, Nginx

### 2. VideoVault
Video management and storage system.

### 3. Payment
Payment processing system.

### 4. VLLM
AI/ML project with image generation capabilities using Stable Diffusion Forge.

**Features:**
- AI Image Generation
- Pinokio integration
- Multiple model support

## Setup

Each project has its own setup instructions. Please refer to individual project directories for specific setup steps.

### General Requirements
- Docker & Docker Compose
- Node.js (for local development)
- PostgreSQL (or use Docker)

## Environment Variables

Copy `.env.example` files to `.env` in each project directory and configure as needed.

## Deployment

Projects are configured for Docker deployment. Use the provided `docker-compose.yml` files.

```bash
docker-compose up -d
```
