## üí° Tips for Maintaining Small Repository

1. **Never commit large files:**
   - Check .gitignore before committing
   - Use `git status` to verify

2. **Regular checks:**
   ```bash
   git count-objects -vH
   ```

3. **If you accidentally commit large files:**
   ```bash
   git rm --cached path/to/large/file
   git commit -m "Remove large file"
   ```

4. **Use Git LFS for unavoidable large files:**
   ```bash
   git lfs install
   git lfs track "*.psd"
   ```
